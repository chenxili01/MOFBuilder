# API Reference (Public Surface)

MofBuilder's detailed API docs are generated from `docs/source/api_reference.rst`.
The curated API focuses on public classes/functions rather than internal helper
modules.

Main public areas:

- Builder and framework orchestration
- Core topology/building components
- I/O readers/writers
- Visualization
- Modelling/simulation utilities

For the rendered API pages, use the Sphinx build output (`docs/_build/html`) or
Read the Docs site.
